# 虚拟试穿结果页 (Try-On Result Page)

**页面标识：** `TryOnResultScreen`
**导航路径：** 方案详情页 → 点击"虚拟试穿" → 试穿生成中 → 试穿结果页
**设计版本：** v1.0
**最后更新：** 2026-01-04

---

## 页面概述

### 页面目标
- **主要目标**：展示 AI 生成的高保真试穿效果，让用户"身临其境"。
- **次要目标**：提供原图对比功能，增强信任感。
- **情感目标**：惊喜感、真实感、科技感。

### 用户任务
1. **主任务**：查看试穿效果图。
2. **次任务**：对比原图，确认效果。
3. **支持任务**：保存图片、分享、重新生成。

---

## 设计方向

**核心视觉特征：**
- **沉浸式全屏**：最大限度展示图片细节。
- **暗色模式基调**：黑色背景，突出主体内容，类似相册浏览体验。
- **Glassmorphism (玻璃态)**：操作栏使用高模糊毛玻璃，悬浮于图片之上，不阻挡视线。

## 组件清单

### 1. 顶部导航 (Transparent Nav)
- **背景**：完全透明
- **返回按钮**：
  - 样式：圆形毛玻璃 `rgba(0,0,0,0.3)`
  - 图标：白色箭头
  - 作用：返回上一页

### 2. 效果图容器 (Image Container)
- **功能**：展示生成的试穿图
- **交互**：
  - 双击：放大/缩小
  - 双指：缩放/平移
- **占位符**：AI 生成完成前的占位（粒子光效）

### 3. 对比开关 (Compare Toggle)
- **样式**：半透明胶囊 `rgba(0,0,0,0.6)`
- **位置**：底部居中，操作栏上方
- **文案**："按住对比原图"
- **交互**：
  - 按住 (Touch Start/Mouse Down)：显示用户上传的原图
  - 松开 (Touch End/Mouse Up)：恢复显示试穿效果图
  - 动画：淡入淡出 (Opacity transition 0.2s)

### 4. 底部操作栏 (Glass Action Bar)
- **样式**：
  - 背景：`rgba(30,30,30, 0.65)`
  - 模糊：`backdrop-filter: blur(25px)`
  - 圆角：32px
  - 边框：1px solid `rgba(255,255,255,0.1)`
  - 阴影：`0 10px 40px rgba(0,0,0,0.4)`
  - 布局：Flex (Space Between)
  - 边距：20px

- **按钮组**：
  - **左侧功能组**：
    - 分享按钮 (Circle Btn) - `share` icon
    - 更多按钮 (Circle Btn) - `more-vertical` icon
  - **主操作按钮**：
    - 保存按钮 (Primary Btn) - "保存到相册"
    - 样式：白色胶囊，黑色文字

## 动画规范

### 1. 魔法生成动画 (Magic Fade)
- **时机**：页面刚进入，代替 Loading 结束的转场
- **效果**：白色/光效遮罩 `opacity: 1 -> 0` 逐渐消失，露出底下的生成图。
- **元素**：Sparkles (粒子) 随机闪烁

### 2. 对比转场
- **效果**：原图层覆盖在结果图层之上。
- **过渡**：`opacity` 0.2s 线性过渡。

---

## 状态定义

### 1. 默认状态
- 展示试穿结果图
- 魔法光效淡出

### 2. 对比状态 (按住)
- 原图不透明度 100%
- 提示文案隐藏或变态
