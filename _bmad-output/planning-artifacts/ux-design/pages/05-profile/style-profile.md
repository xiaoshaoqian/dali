# 风格档案 (Style Profile Page)

**页面标识：** `StyleProfileScreen`
**导航路径：** 我的页面 → 风格档案
**设计版本：** v1.0
**最后更新：** 2026-01-14

---

## 页面概述

### 页面目标
- **主要目标**：展示 AI 根据用户历史搭配和偏好分析出的个人风格画像。
- **视觉目标**：使用标签云和进度条直观展示风格成分，延续紫色渐变头部设计。

### 功能范围
- 展示个人风格关键词（标签云形式）
- 展示风格成分分析（进度条形式）
- 提供重新测试入口

---

## 设计方向

**核心视觉特征：**
- **Purple Gradient Header**: 紫色渐变导航栏，包含返回按钮和重新测试按钮
- **White Content Card**: 白色内容卡片上浮覆盖头部
- **Tag Cloud**: 多尺寸标签云展示风格关键词
- **Progress Bars**: 彩色进度条展示风格成分占比

## 数据结构

### 风格关键词 (Keywords)
```typescript
interface StyleKeyword {
  text: string;      // 关键词文字
  weight: 'primary' | 'medium' | 'default'; // 权重等级
}

// 示例数据
const keywords: StyleKeyword[] = [
  { text: '简约通勤', weight: 'primary' },
  { text: '韩系温婉', weight: 'default' },
  { text: '舒适休闲', weight: 'default' },
  { text: '低饱和度', weight: 'default' },
  { text: '基础款', weight: 'default' },
];
```

### 风格成分 (Style Composition)
```typescript
interface StyleComponent {
  name: string;      // 风格名称
  percentage: number; // 占比 (0-100)
  color: string;     // 进度条颜色
}

// 示例数据
const components: StyleComponent[] = [
  { name: '简约', percentage: 45, color: '#6C63FF' },
  { name: '优雅', percentage: 30, color: '#FF9500' },
  { name: '休闲', percentage: 15, color: '#34C759' },
  { name: '其他', percentage: 10, color: '#8E8E93' },
];
```

## 交互定义

| 触发元素 | 交互行为 | 目标 |
|---------|---------|------|
| 返回按钮 | 点击 → 返回上一页 | Profile Page |
| 重新测试 | 点击 → 启动风格测试问卷 | Style Quiz Flow |
| 标签 | 点击 → 展示风格说明 (可选) | Tooltip/Modal |

---

## 状态定义

| 状态 | 视觉表现 |
|-----|---------|
| 数据已加载 | 显示标签云和进度条 |
| 数据为空 | 显示空状态引导重新测试 |
| 加载中 | 骨架屏动画 |

---
