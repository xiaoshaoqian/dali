# 身材档案 (Body Profile Page)

**页面标识：** `BodyProfileScreen`
**导航路径：** 我的页面 → 身材档案
**设计版本：** v1.0
**最后更新：** 2026-01-14

---

## 页面概述

### 页面目标
- **主要目标**：集中展示和管理用户的身材数据，为 AI 搭配推荐提供基础信息。
- **视觉目标**：延续紫色渐变头部和白色卡片的层叠设计，保持全局一致性。

### 功能范围
- 展示基本身材信息（性别、身高、体重、体型）
- 展示外貌特征（脸型、发型、发色）
- 展示尺码参考（上装、下装、鞋码）
- 提供编辑入口

---

## 设计方向

**核心视觉特征：**
- **Purple Gradient Header**: 紫色渐变导航栏，包含返回按钮和编辑按钮
- **White Content Card**: 白色内容卡片上浮覆盖头部 (24px radius, margin-top: -30px)
- **Section Tags**: 分组标题带有必填/选填/参考标签
- **List-style Layout**: iOS 风格列表布局，键值对展示

## 组件清单

### 1. 紫色渐变头部 (Header Gradient)
- **背景**: Linear Gradient (`#6C63FF 0%` → `#8578FF 100%`)
- **高度**: padding-top: 59px + content
- **导航元素**:
  - 左侧: 返回按钮 (白色箭头)
  - 中间: 页面标题 "身材档案" (17px, Semibold, 白色)
  - 右侧: 编辑按钮 (17px, 白色)

### 2. 白色内容卡片 (Content Card)
- **背景**: 白色 (#FFFFFF)
- **圆角**: 24px 24px 0 0
- **位置**: margin-top: -30px (上浮覆盖)
- **内边距**: 24px 20px

### 3. 分组标题 (Section Title)
- **布局**: Flex Row (标题 + 标签)
- **标题样式**: 13px, #8E8E93, margin-bottom: 10px
- **标签类型**:
  | 类型 | 颜色 | 背景 |
  |-----|------|------|
  | 必填 | #6C63FF | rgba(108,99,255,0.1) |
  | 选填 | #8E8E93 | rgba(142,142,147,0.15) |
  | 参考 | #6C63FF | rgba(108,99,255,0.1) |

### 4. 数据卡片 (Card)
- **背景**: 白色
- **圆角**: 20px
- **边框**: 1px solid rgba(0,0,0,0.02)
- **阴影**: 0 2px 12px rgba(0,0,0,0.04)

### 5. 列表项 (List Item)
- **高度**: min-height: 52px, padding: 16px 20px
- **布局**: Flex Row (标签 - Spacer - 值)
- **标签**: 16px, #1C1C1E
- **值**: 16px, #8E8E93
- **未填写**: 16px, #C7C7CC, 显示 "未选择" + 箭头
- **分割线**: 0.5px, rgba(0,0,0,0.05)

## 数据结构

### Section 1: 基本信息 (必填)
| 字段 | 标签 | 数据类型 | 示例值 |
|-----|------|---------|-------|
| gender | 性别 | Enum | 女 / 男 |
| height | 身高 | Number + Unit | 165 cm |
| weight | 体重 | Number + Unit | 48 kg |
| bodyType | 体型 | Enum | 梨形身材 |

### Section 2: 外貌特征 (选填)
| 字段 | 标签 | 数据类型 | 示例值 |
|-----|------|---------|-------|
| faceShape | 脸型 | Enum | 鹅蛋脸 / 圆脸 / 方脸 |
| hairStyle | 发型 | Enum | 长卷发 / 短发 / 中长发 |
| hairColor | 发色 | Enum | 深棕色 / 黑色 / 浅棕色 |

### Section 3: 尺码参考 (参考)
| 字段 | 标签 | 数据类型 | 示例值 |
|-----|------|---------|-------|
| topSize | 上装 | String | S / 160 |
| bottomSize | 下装 | String | M / 27 |
| shoeSize | 鞋码 | Number | 37 |

## 交互定义

| 触发元素 | 交互行为 | 目标 |
|---------|---------|------|
| 返回按钮 | 点击 → 返回上一页 | Profile Page |
| 编辑按钮 | 点击 → 进入编辑模式 | Body Profile Edit |
| 未填写项 | 点击 → 进入选择器 | 对应 Picker |

---

## 状态定义

| 状态 | 视觉表现 |
|-----|---------|
| 已填写 | 值为深灰色 (#8E8E93) |
| 未填写 | 值为浅灰色 (#C7C7CC)，显示 "未选择" + 箭头 |
| 编辑模式 | 所有项可点击，显示选择器 |
